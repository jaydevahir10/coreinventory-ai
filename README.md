# CoreInventory AI

CoreInventory AI is an AI-powered inventory management system designed to help warehouses track stock, detect low inventory, and receive intelligent restock suggestions.

## Features

- Dashboard overview
- Product inventory tracking
- Low stock detection (AI insights)
- Backend API with database
- Responsive UI

## Tech Stack

Frontend: React  
Backend: Node.js + Express  
Database: SQLite  

## Project Structure

coreinventory-ai
│
├ backend
│   ├ server.js
│
├ frontend
│   ├ src
│
└ README.md

## How to Run

### Backend

cd backend  
npm install  
node server.js  

### Frontend

cd frontend  
npm install  
npm start
